# remark-flow

用于在 React 应用中解析和处理 MarkdownFlow 语法的 remark 插件。扩展 react-markdown 支持变量处理和 AI 集成功能。

## 概述

remark 插件为 react-markdown 提供 MarkdownFlow 语法支持，在现有 markdown 工作流中实现个性化内容渲染。

## 安装

```bash
npm install remark-flow
# 或
yarn add remark-flow
```
