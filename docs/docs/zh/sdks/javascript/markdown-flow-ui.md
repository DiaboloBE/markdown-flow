# markdown-flow-ui

用于渲染交互式 MarkdownFlow 文档的 React 组件库，包含内置 UI 元素。提供即用组件，用于创建个性化的交互式内容体验。

## 概述

UI 库提供具有内置样式和功能的完整 React 组件。非常适合需要全功能 MarkdownFlow 渲染的应用程序，设置最少。

## 安装

```bash
npm install markdown-flow-ui
# 或
yarn add markdown-flow-ui
```
