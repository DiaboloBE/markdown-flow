markdown-flow-ui
前端
具有内置功能的完整 UI 组件。适用于交互式 Markdown 文档的即用型 React 组件。
remark-flow
前端
用于 MarkdownFlow 语法的 React-markdown 插件。扩展 react-markdown 支持变量处理和 AI 集成。
markdown-it-flow
前端
用于 MarkdownFlow 语法的 Markdown-it 插件。为任何基于 markdown-it 的应用程序添加 MarkdownFlow 功能。
